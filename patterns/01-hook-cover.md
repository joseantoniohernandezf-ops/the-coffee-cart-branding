# Pattern 01 — Hook / Portada · The Coffee Cart

El slide más importante. Si no engancha en 1 segundo, no hay carrusel.

## Fórmula

```
[Fondo]         Café oscuro (#5C3220) o Crema (#F5F2EB)
[Capa texto]    2 tipografías: Avenir Black XXL + Lora Italic de apoyo
[Visual]        Foto de bebida, barra o momento de evento (con overlay)
[Dots]          Mockup de dots de carrusel en la parte inferior
```

## Estructura visual

```
┌─────────────────────────────────┐
│                                 │
│   [Lora Italic crema/café]      │  ← "El detalle que" / "Así se ve"
│                                 │
│   [KEYWORD XXL BLANCO/CREMA]    │  ← palabra central del hook
│   [KEYWORD XXL BLANCO/CREMA]    │     Avenir Black, ocupa 60% del ancho
│                                 │
│      [foto/bebida con overlay]  │  ← imagen con overlay café o negro
│                                 │
│   [Avenir Regular pequeño]      │  ← subtítulo corto que promete valor
│                                 │
│    ● ● ● ● ○ ○ ○ ○              │  ← dots de carrusel
└─────────────────────────────────┘
```

## Specs técnicas

| Elemento | Detalle |
|---|---|
| Ratio | 4:5 (1080 × 1350 px) |
| Fondo opción A | Café `#5C3220` sólido |
| Fondo opción B | Crema `#F5F2EB` + foto con overlay negro `rgba(13,13,13,0.55)` |
| Lora Italic | 70-90pt, crema `#F5F2EB` o blanco `#FFFFFF` |
| Keyword XXL | Avenir Black / Nunito 900, 160-220pt, blanco `#FFFFFF` o crema |
| Subtítulo | Avenir Regular 26-32pt, blanco o crema |
| Dots | Mini círculos blancos, 12-14px diámetro, abajo centrado |

## Variantes

### Variante A — Aspiracional (la más usada)
```
Lora Italic:   "El detalle que"
Keyword XXL:   TUS INVITADOS
               RECUERDAN
Sub:           Café de especialidad para tu evento →
```

### Variante B — Provocación / Pregunta
```
Keyword XXL:   ¿CAFÉ DE
               MÁQUINA?
Lora Italic:   "tu evento merece más"
Sub:           Desliza y descubre la diferencia
```

### Variante C — Dato / Resultado
```
Lora Italic:   "Más de 50 eventos"
Keyword XXL:   SERVIDOS
               CON CAFÉ
Sub:           de especialidad · The Coffee Cart
```

## Reglas obligatorias

1. **Máximo 2 familias tipográficas** por slide (Avenir + Lora)
2. **Una keyword grande** en blanco o crema — nunca en café sobre café
3. **Siempre con imagen o fondo de color sólido** — nunca fondo blanco puro en portada
4. **Dots de carrusel** visibles abajo
5. **Lora Italic siempre en minúsculas** — es el toque elegante

## Qué evitar

* ❌ Fondo blanco puro `#FFFFFF` en portada
* ❌ Keyword en color café sobre fondo café (sin contraste)
* ❌ Más de 3 tamaños de texto distintos
* ❌ Foto sin overlay (el texto debe ser legible siempre)
* ❌ Mencionar precios en portada

## Prompt para Claude / IA

> Portada de carrusel vertical 4:5 para Instagram. Fondo café oscuro #5C3220 o foto de evento con overlay rgba(13,13,13,0.55). Título en dos niveles: frase corta de apoyo en Lora Italic blanco crema #F5F2EB 80pt arriba, luego la keyword principal en Avenir Black / Nunito 900 blanco ULTRA grande 200pt ocupando 60% del ancho. Subtítulo pequeño Avenir Regular 28pt crema debajo. Dots de carrusel blancos al pie. Estilo elegante, limpio, profesional.
