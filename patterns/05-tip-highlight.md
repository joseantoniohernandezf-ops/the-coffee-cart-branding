# Pattern 05 — Tip destacado

Slide con un consejo práctico sobre café o eventos. Posiciona a The Coffee Cart como experto.

## Fórmula

```
[Fondo]        Crema (#F5F2EB) o Blanco (#FFFFFF)
[Visual]       Foto grande de bebida o proceso (close-up)
[Etiqueta TIP] "TIP" en Lora Italic grande + número si es serie
[Flecha]       Separador o flecha hacia el consejo
[Consejo]      Avenir Black para la idea principal + Avenir Regular para el detalle
```

## Estructura visual

```
┌─────────────────────────────────┐
│                                 │
│   ┌─────────────────────────┐   │
│   │                         │   │
│   │   FOTO CLOSE-UP         │   │  ← bebida, latte art, granos,
│   │   (café o evento)       │   │    manos del barista, vapor
│   │                         │   │
│   └─────────────────────────┘   │
│                                 │
│  [TIP] Lora Italic →            │  ← "TIP →" en café #5C3220
│                                 │
│  [CONSEJO PRINCIPAL]            │  ← Avenir Black negro, 36-44pt
│  [detalle en Avenir Regular]    │  ← Avenir Regular negro, 22-26pt
│                                 │
└─────────────────────────────────┘
```

## Specs

| Elemento | Detalle |
|---|---|
| Ratio | 4:5 |
| Fondo | Crema `#F5F2EB` |
| Foto | 70-75% ancho, centrada arriba, sin filtro o con tono cálido sutil |
| "TIP" | Lora Italic 72-90pt, café `#5C3220` |
| Consejo principal | Avenir Black 36-44pt, negro tinta `#0D0D0D` |
| Detalle | Avenir Regular 22-26pt, negro tinta |

## Ejemplos de tips reales para The Coffee Cart

### Serie: Tips de café de especialidad
* **TIP 01** — "El café de especialidad tiene puntaje. Busca granos con 85+ puntos SCA."
* **TIP 02** — "El espresso tiene 3 fases: pre-infusión, extracción y corte. Las tres importan."
* **TIP 03** — "El agua representa el 98% de tu taza. La calidad del agua cambia todo."
* **TIP 04** — "El cold brew no es café frío. Es café extraído en frío por 12-24 horas."

### Serie: Tips para organizar tu evento
* **TIP 01** — "Planea 1 barista por cada 80-100 invitados para un servicio sin colas."
* **TIP 02** — "El café de bienvenida reduce el tiempo de espera en registro."
* **TIP 03** — "Ofrece opciones sin cafeína — el 20% de tus invitados las necesitará."

### Tips aspiracionales (para engagement)
* **TIP** — "Un buen café al final de la cena es lo que hace que la gente se quede."
* **TIP** — "El café de especialidad en tu evento dice más de ti que el decorado."

## Variantes de visual

* **Close-up de latte art** → tip sobre espresso o presentación
* **Close-up de granos** → tip sobre origen o puntaje
* **Foto de barista trabajando** → tip sobre proceso o extracción
* **Foto del setup en evento** → tip sobre logística o planeación
* **Grid 2x2 de bebidas** → tip comparativo (espresso vs americano, etc.)

## Reglas

1. **"TIP" siempre en Lora Italic** — distingue este formato de los pasos numerados
2. **Foto siempre arriba** — el visual primero, el consejo al leer
3. **Consejo en máximo 2 líneas** de Avenir Black
4. **Detalle opcional** en Avenir Regular debajo — si el tip necesita contexto

## Prompt para Claude / IA

> Slide 4:5. Fondo crema #F5F2EB. En la mitad superior, una foto close-up de alta calidad de [elemento del tip] ocupando 70% del ancho, con filtro cálido sutil (temperatura +15). Debajo de la foto, la palabra "TIP" en Lora Italic color café #5C3220 80pt seguida de una flecha →. Debajo, el consejo principal en Avenir Black negro tinta 40pt en 1-2 líneas. Si hay detalle, en Avenir Regular negro tinta 24pt. Mucho espacio en blanco, estilo editorial limpio.
