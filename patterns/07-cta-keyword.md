# Pattern 07 — CTA ("Cotiza tu evento")

Último slide. El que convierte. Pide al usuario que tome acción concreta.

## Fórmula

```
[Fondo]            Negro tinta (#0D0D0D) + patrón C o granos en opacidad 10%
[Acción]           "COTIZA" o "ESCRÍBENOS" en Avenir Black crema arriba
[Keyword/CTA]      Frase de acción en Lora Italic crema XXL, centrada
[Promesa]          Texto en crema/blanco explicando qué reciben
[Footer]           Logo + @handle en crema al pie
```

## Estructura visual

```
┌─────────────────────────────────┐
│                                 │
│       COTIZA TU EVENTO          │  ← Avenir Black crema 80-100pt
│                                 │
│   "el café que se recuerda"     │  ← Lora Italic crema/latte XXL
│                                 │
│                                 │
│    ✓ Sin costo de traslado      │  ← Avenir Regular blanco 32pt
│    ✓ Menú personalizado         │
│    ✓ Respuesta en 24 horas      │
│                                 │
│    [THE COFFEE CART logo]       │  ← logo blanco/crema centrado
│       @thecoffeecart            │  ← handle crema al pie
└─────────────────────────────────┘
```

## Specs

| Elemento | Detalle |
|---|---|
| Ratio | 4:5 |
| Fondo | Negro tinta `#0D0D0D` + patrón (C o granos) al 10% de opacidad |
| "COTIZA TU EVENTO" | Avenir Black crema `#F5F2EB` 80-100pt, centrado arriba |
| Lora Italic | Crema `#F5F2EB` o latte `#9E7B65`, 90-120pt, centrada |
| Checks ✓ | Color latte `#9E7B65` + Avenir Regular blanco 32pt |
| Logo | Versión blanca/crema, centrado |
| @handle | Avenir Medium crema 28-34pt, centrado al pie |

## Criterios para el copy del CTA

* **Verbo de acción claro** (Cotiza, Escríbenos, Agenda) — nunca pasivo
* **Lora Italic** siempre en minúsculas y entre comillas — toque de elegancia
* **Máximo 3 checks** — si son más, edita
* **Checks concretos** — beneficios tangibles, no frases genéricas
* **@handle siempre presente** — es la puerta de entrada

## Reglas obligatorias

1. **Fondo negro tinta** — contrasta con todos los slides anteriores, señala el cierre
2. **Patrón de marca al 10%** — textura sutil que recuerda la identidad
3. **Logo siempre presente** — último recuerdo de marca
4. **@handle siempre al pie** — sin excepciones
5. **Lora Italic para la frase principal** — coherente con el tono elegante
