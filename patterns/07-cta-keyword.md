# Pattern 07 — CTA ("COMENTA '[KEYWORD]'")

Último slide. El que convierte. Pide al usuario que comente una palabra clave para recibir algo por DM.

## Fórmula

```
[Fondo]            Azul primario sólido + textura papel
[Verbo]            "COMENTA" en sans bold condensed blanco arriba
[Palabra clave]    Entre comillas, amarillo acento XXL, centrada
[Promesa]          Texto en blanco explicando qué recibe
[Footer]           @tu-handle en blanco al pie
```

## Estructura visual

```
┌─────────────────────────────────┐
│                                 │
│       COMENTA                   │  ← sans bold condensed blanco 100pt
│                                 │
│       "[KEYWORD]"               │  ← sans bold condensed AMARILLO XXL
│                                 │     (entre comillas)
│                                 │
│    [promesa línea 1]            │  ← sans bold condensed blanco 40pt
│    [promesa línea 2]            │     (distribuido, texto corto)
│    [promesa línea 3]            │
│    [promesa línea 4]            │
│                                 │
│                                 │
│       @tu-handle                │  ← sans bold blanco 36pt (footer)
└─────────────────────────────────┘
```

## Specs

| Elemento | Detalle |
|----------|---------|
| Ratio | 4:5 |
| Fondo | Azul primario `#9BBDDC` + papel arrugado textura |
| "COMENTA" | Sans bold condensed blanco roto 100-140pt, centrado arriba |
| Palabra clave | Sans bold condensed amarillo acento `#E8C547` 200-280pt, entre comillas, centrada |
| Promesa | Sans bold condensed blanco 40-50pt, distribuido en 4 líneas cortas |
| Footer @handle | Sans bold blanco 32-40pt, centrado al pie |

## Criterios para una buena keyword

- **Corta** (1 palabra ideal, máximo 2)
- **Fácil de escribir** (sin acentos raros)
- **Relacionada con el tema del post**
- **Genera curiosidad** ("qué me mandará si comento esto")

## Variantes

### Variante A — Keyword corta (1 palabra)
```
COMENTA
"[KEYWORD]"
y te mando [cosa concreta]
```

### Variante B — Keyword + promesa de estrategia
```
COMENTA
"[KEYWORD]"
y te mando mi [estrategia/plantilla/recurso]
```

## Reglas obligatorias

1. **Siempre entre comillas** la keyword
2. **Amarillo acento siempre** para la keyword
3. **Azul primario siempre** como fondo
4. **@handle footer** sin excepciones
5. **Texto de promesa corto** — máximo 4 líneas de 4-6 palabras cada una
